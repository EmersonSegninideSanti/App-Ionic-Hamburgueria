# App-Ionic-Hamburgueria
Projeto de aplicativo de restaurante para o curso Desenvolvimento para Dispositivos Móveis. Aplicativo Ionic com framework Angular





## Forma do aplicativo na versão 1 para entrega

### Página inicial
• Logo **R01**

• Ícone de carrinho (não funcional)(ou sacola) **R02**

• Prato do dia (Letreiro "Prato do dia", Foto, nome, preço) **R03**

• Links para as páginas de categoria (devem ser chamativos. Foto grande e texto abaixo) **R04**

### Página de categoria

• 4 páginas de categorias (1 lanches, 2 sobremesas, 3 entradas, 4 entradas e pratos de refeição) **R05**

• Itens dos alimentos da categoria (constituído por ícone, nome e preço) **R06**

### Página de produto


• 5 páginas estáticas para cada categoria **R07**

• Conteúdo: Foto, nome, descrição (ingredientes) e preço **R08**

### Aparência

• Cores: Primary (bege e branco), Secondary(branco e preto), tertiary(marrom e branco) **R09**

• Fonte: Roboto (Textos destacados devem ser máximamente destacados) **R10**
